# Job Portal Backend (Spring Boot)

This project is a scalable backend system for a Job Portal application built using Java and Spring Boot. It provides RESTful APIs for managing users, job postings, and applications with secure authentication and efficient database handling.

## 🚀 Features
- User registration and login with JWT authentication
- Role-based access control (Admin/User)
- Job posting and management
- Job application tracking system
- Secure REST APIs
- Layered architecture (Controller, Service, Repository)

## 🛠️ Tech Stack
- Java
- Spring Boot
- Spring Security (JWT)
- MySQL
- Maven

## 📂 Project Structure
- Controller Layer – Handles API requests
- Service Layer – Business logic
- Repository Layer – Database operations

## 🔐 Security
- JWT-based authentication
- Password encryption
- Protected API endpoints

## 📌 Use Case
This project simulates a real-world job portal backend used by recruiters and job seekers.
